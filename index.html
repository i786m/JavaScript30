JavaScript30

JavaScript30 is 30 day coding challenge to build a project per day for a month. The challenge revolves around the dominant front end languages that are used to create the client side view of internet websites, HTML for the content, CSS for the styling and Javascript for the interactive behaviour of websites.

I will be up loading the links to each of the projects here daily.

  • Day 1 : JavaScript Drum Kit
  • The task for day one was to build a simple drum kit. Using the keyboard you can simulate a working drumkit. Be warned though, it can get pretty addictive playing around with it!
  • Day 2 : JS & CSS clock
    •
  • The task for day two was to build wprking analog clock. Using JS and CSS to get the hands working and rotating around the clock face.
